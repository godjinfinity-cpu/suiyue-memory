红灯牌收音机 20世纪80年代 家中客厅

这件物品的故事

这台收音机是父亲当年省下一个月工资买回来的。每天傍晚，家里人都会围坐在一起，听新闻、听评书、听音乐。它的声音有时并不清楚，却把一家人聚在了一张桌旁。后来电视和手机慢慢多了，这台收音机被收进柜子，但它留下的热闹和陪伴，一直没有被忘记。
