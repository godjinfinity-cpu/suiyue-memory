上海牌手表 1950年代 上海南京东路

这件物品的故事

这只上海牌手表，是我年轻时在南京东路买下来的。那时候工资不高，我攒了好久才舍得把它带回家。它陪我上班、结婚、生子，也陪着一家人度过了许多平常却珍贵的日子。现在它已经不再走了，但每次拿起来，我还是会想起那个认真生活、一步一步把日子过好的年代。

它记录的不只是时间，也是我们一家人的日子。
